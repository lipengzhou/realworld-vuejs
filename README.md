# RealWorld demo app

> A RealWorld demo app by Vue.js

## 技术栈

- Vue.js
- Vue Router
- Vue CLI
- Vuex

## 功能

- [ ] 首页
  + [ ] 展示标签列表
  + [ ] 展示文章列表
  + [ ] 文章列表分页
- [ ] 用户注册 / 登录
  + [ ] 使用 JWT（将 token 放到本次存储）
  + [ ] 身份验证可以切换到基于 session/cookie 机制
- [ ] 用户设置
- [ ] 创建文章
- [ ] 编辑文章
- [ ] 文章详情
  + [ ] 删除文章
  + [ ] 文章评论
  + [ ] 删除评论
- [ ] 个人资料
  + [ ] 展示用户基本信息
  + [ ] 展示用户文章列表
  + [ ] 展示用户点赞的文章
